# 项目号

| 标签 | Project |
| --- | --- |
| 最后编辑时间 | 2025/07/08 10:15 |
- [合同项目](%E5%90%88%E5%90%8C%E9%A1%B9%E7%9B%AE%20296f8e21a0254cf18156fe5db5635a39.md)的项目号，由[业务类型](%E4%B8%9A%E5%8A%A1%E7%B1%BB%E5%9E%8B%20906718c19c174d3cbf3bc9ef51eb6567.md)字母+年份（2位）+流水号（3位）组成。
    - 可带“-”和四位数字的尾缀，表示[工作单](%E5%B7%A5%E4%BD%9C%E5%8D%95%20314cc4c4505140df85bfb820271e450e.md)
    - 可带“ _”和字母的尾缀，表示EPC的拆分
    - 带三位数字或字母加数字的，为历史遗留问题。
- [部门项目](%E9%83%A8%E9%97%A8%E9%A1%B9%E7%9B%AE%2004032a43ced445bea304d39d15a387f9.md)的项目号可以视为部门号，通常以SAS开头
    - 可带“-”和四位数字的尾缀，表示部门下的业务
