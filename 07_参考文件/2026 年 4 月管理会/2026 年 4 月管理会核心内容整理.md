# 2026 年 4 月管理会核心内容整理

## 一、2026 考核机制：三大体系

公司 2026 年考核体系分为三大类：**共赢合伙（项目交付/业务板块）**、**成本中心（专业室）**、**职能部门**。

### 1. 共赢合伙机制（项目交付板块）
> **定位**：利润中心。绑定目标，激励长效增长，收益与风险对等。

#### 1.1 基本原则
*   **共享范围**：板块/中心与公司共享年度利润贡献。
*   **人员分类**：
    *   **共享伙伴**：项目群负责人、板块/中心负责人、项目经理及以上管理团队。
    *   **非共享伙伴**：工程师团队（含设计咨询类）、管理辅助人员、现场派驻工程师。
*   **核算基准**：以**开票收入 (R)** 为基准（含内部收入，需剔税）。
*   **岗位工资暂缓**：共享伙伴的岗位工资年度内**暂缓发放**，年底根据利润贡献度 (P) 情况发放。

#### 1.2 核算公式与梯级
*   **利润贡献值 (P)**：`P = 年度开票收入 R – 年度总实际发生成本 - 公司管理费 OH`
*   **可分配奖金 (B)**：`B = P * X%`（X%为利润共享比例）
*   **共享比例梯级**（根据 `Q = 实际开票 / 开票指标`）：
    *   `Q ≤ 0.875`：共享比例 **35%**
    *   `0.875 < Q < 1.125`：按**实际比例**共享
    *   `Q ≥ 1.125`：共享比例 **45%**
*   **公司管理费标准**：设计咨询类 **9%**，大型 EPC **1.5%**，小型 EPC **3%**，平进平出 **0.5%**。

#### 1.3 兑现条件
*   **共享伙伴**：
    *   **前提**：`P > 0` 且 **现金流为正**。
    *   **发放**：按照 `(年度开票对应的回款 / 年度开票)` 的比例发放奖金。
    *   **节奏**：次年第二季度起每季度核算一次并发放；现金流为负待转正后发放。
    *   **亏损分担**：若 `P < 0`，无奖金；按 **40%:60%** 比例与公司共同承担亏损。如有余额，次年年初发放。
*   **工程师团队**：
    *   **发放**：按**年度**发放，由板块/中心自行分配。
    *   **资格**：个人内部收入 > 个人成本的工程师，具备在板块亏损时由公司统一分配奖金的资格。
    *   **工时改革**：取消工程师填报工时 **49小时/周** 的上限，所有确认的项目/报价工时计入项目成本。

#### 1.4 否决项/减分项
*   **一票否决**：发生三级以上质量或安全事故、道德事件、合规事件。
*   **发票红冲**：等额损失（税）从共享伙伴奖金中扣除。
*   **坏账**：全部计入成本，并按额度分级扣除奖金。

### 2. 成本中心考核（专业室）
> **定位**：成本控制、效率提升。不考核利润，重点考核成本、费用控制及交付质量/效率。
> **范围**：五大专业室（工艺设备、配管、仪电、土建、项目管理服务）、生态合作部采购工程师团队。

#### 2.1 核算原则
*   **内部收入 (N)**：`N = ∑(工程师费率 R1 * 工时 MH * (1 + 奖金系数 Y%))`
*   **实际成本 (C)**：`C = ∑ 工程师个人成本 C1`
*   **可分配奖金 (B)**：`B = N – C`
*   **分配规则**：
    *   `B > 0`：可进行分配，由专业室主任根据工时和贡献度分配。
    *   `B < 0`：**室主任无奖金**。
    *   **工程师保护**：若工程师个人 `N1 - C1 > 0`，即使专业室整体亏损，该工程师仍可根据公司运营情况获得奖金。

#### 2.2 兑现与否定
*   **工时改革**：同样取消 49小时/周上限，监测异常填报。
*   **一票否决**：三级以上质量/安全事故、道德/合规事件。

---

### 3. 职能部门考核
> **范围**：五大职能部门（不含已划入成本中心的人员）、技术委员会。

#### 3.1 核算原则
*   **奖金总额**：`部门奖金总额 = ∑(个人奖金基数 * 部门综合 KPI)`
*   **KPI 评定**：由 P&L 板块/中心、专业室及职能部门之间**综合评定**。
*   **岗位职级**：由部门提出，人力资源部根据实际工作内容和能力确认。

#### 3.2 兑现条件
*   **公司 EBIT > 0**：可分配奖金，由部门负责人自行分配。
*   **公司 EBIT < 0**：**无奖金**。

## 二、利润共享与岗位工资模拟测算（Excel 数据转列表）

以下数据为模拟测算示例，展示不同场景下的奖金与工资发放逻辑。

### Sheet 1: 利润共享奖金模拟测算（设计咨询类业务）

- **场景 A：超额完成 (Q > 1.125)**
  - **开票总额**：8,200 万 | **开票指标**：7,200 万 | **Q 值**：1.14
  - **利润贡献 (P)**：770 万 | **共享比例**：45% | **可分配奖金 (B)**：346.5 万
  - **回款情况**：回款 8,000 万（回款/开票 = 97.5%）
  - **发放结果**：次年 Q1 可发 338 万，余额 8.4 万待后续回款分配。

- **场景 B：达标完成 (0.875 < Q < 1.125)**
  - **开票总额**：4,000 万 | **开票指标**：4,300 万 | **Q 值**：0.93
  - **利润贡献 (P)**：200 万 | **共享比例**：37.2%（实际比例） | **可分配奖金 (B)**：74.4 万
  - **回款情况**：回款 3,600 万（回款/开票 = 90%）
  - **发放结果**：次年 Q1 可发 67 万，余额 7.4 万。次年 Q2 回款 200 万，可再发 3.7 万。

- **场景 C：亏损情况 (P < 0)**
  - **开票总额**：800 万 | **利润贡献 (P)**：-120 万
  - **结果**：无可分配奖金。

### Sheet 2: 共享伙伴岗位工资发放模拟计算

- **情形 A：盈利且现金流为正**
  - **条件**：`P > 0`，次年 Q1 现金流 `> 0`
  - **规则**：全额发放岗位工资。
  - **示例**：年工资 144 万，Q1 实发 144 万。

- **情形 B：盈利但现金流为负（缓发）**
  - **条件**：`P > 0`，次年 Q1 现金流 `≤ 0`
  - **规则**：缓发，待现金流转正后发放。
  - **示例**：年工资 120 万，Q1 实发 0；次年 Q2 现金流转正，补发 120 万。

- **情形 C：亏损但现金流为正（按比例发）**
  - **条件**：`P < 0`，次年 Q1 现金流 `> 0`
  - **规则**：按比例发放（公式：`P*40% + 工资总额`，即扣除亏损分担后发放）。
  - **示例**：年工资 60 万，亏损 50 万，Q1 实发 40 万。

- **情形 D：亏损且现金流为负（按比例缓发）**
  - **条件**：`P < 0`，次年 Q1 现金流 `≤ 0`
  - **规则**：按比例缓发。
  - **示例**：年工资 36 万，亏损 16 万，Q1 实发 0；次年 Q2 现金流转正，补发 29.6 万。

## 三、管理岗位人员迭代机制

> **目的**：建立“能上能下、优胜劣汰”的良性迭代机制。
> **适用范围**：P&L 板块/中心、职能部门、专业室、专项委员会负责人（公司正式任命的正副职）。

### 1. 考核维度与处理方式

| 考核维度 | 量化标准/等级 | 首次处理方式 | 第二次出现/连续处理 |
| :--- | :--- | :--- | :--- |
| **单位 KPI (K1)** | K1 < 60 分 | **次年免职** | - |
| | 60 ≤ K1 < 80 分 | **观察留任**，提出改进计划 | 连续两年 < 80 分，**第三年免职** |
| **客户投诉** | 严重 | **当场免职** | - |
| | 一般 | 处分留观（书面警告+6个月） | **当场免职** |
| | 轻微 | 书面警告 | 处分留观（书面警告+6个月） |
| **安全/质量/合规/道德** | 3级及以上事故 | **当场免职** | - |
| | 2级事故 | 处分留观（书面警告+6个月） | **当场免职** |
| | 1级事故 | 书面警告 | 处分留观（书面警告+6个月） |
| **多维度评分 (K)** | K < 60 分 | **次年免职** | - |
| | 60 ≤ K < 80 分 | **观察留任**，提出改进计划 | 连续两年 < 80 分，**第三年免职** |

*注：多维度评分包括直线领导/公司管理者 (K2)、同级/相关部门负责人 (K3)。*

### 2. 申诉机制
*   **时限**：收到处理决定后 **10 个工作日**内。
*   **范围**：对认定、得分存异议等。
*   **要求**：提交书面证据与说明，由公司组织评判。
*   **惩戒**：恶意申诉者给予书面警告并计入考核。
