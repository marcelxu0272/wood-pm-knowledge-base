# 项目管理

> 整合自 04_概念术语 下所有项目相关文档，涵盖项目分类、编码规则、业务类型、工时计算等。

## 项目分类

平台中的**项目**主要可分为以下几类：

| 类型 | 说明 |
| --- | --- |
| **合同项目** | 正式签署合同后的项目，有正式项目号 |
| **部门项目** | 各部门内部业务项目，项目号可视为部门号（通常以 SAS 开头） |
| **协作项目** | 历史遗留的"Workshare 项目"，现已暂停执行 |
| **未签项目** | CRB3 审批通过但尚未正式签署合同前的项目，项目号带部门号 |
| **前期项目** | 同未签项目，用于承载前期项目号阶段的项目执行 |

---

## 项目号编码规则

### 合同项目

项目号由 **业务类型字母** + **年份（2位）** + **流水号（3位）** 组成。

- 可带 `-` 和四位数字的尾缀，表示**工作单**
- 可带 `_` 和字母的尾缀，表示 EPC 的拆分
- 带三位数字或字母加数字的，为历史遗留问题

### 部门项目

项目号可视为部门号，通常以 `SAS` 开头。
- 可带 `-` 和四位数字的尾缀，表示部门下的业务

## 工时成本核算

### 项目计算工时

计算工时根据项目签署的去税合同额与**综合基准费率**计算，用于指导项目报价及设定计划工时（预算工时）。

**一般设计咨询项目公式：**

$$计算工时 = (项目设计咨询部分合同金额 \times 项目人工成本比例) \div 板块综合基准费率$$

### 工时成本占比

$$工时成本占比 = (预计总工时数 \times 固定费率) \div 预估设计费或合同额 \times 100\%$$

**颜色提示逻辑：**

| 占比区间 | 提示色 | 含义 |
| --- | --- | --- |
| < 20% | 红色 | 成本占比过低，需关注 |
| 20% ~ 50% | 绿色 | 正常区间 |
| > 50% | 红色 | 成本占比过高，需关注 |

## Workshare 协作

**Workshare** 是项目协作的一种模式，历史遗留的**协作项目**是其主要体现形式，现已暂停执行。

**Workshare 目标：**

- 降低 Wood China 项目执行的整体成本，增强市场竞争力
- 最大限度发挥技术骨干、资深员工的能力
- 增强 HVEC 的项目执行能力，壮大 HVEC 具有丰富设计经验的人力资源规模
- 提升全公司范围内项目执行标准和管理流程的规范性、一致性和可预见性
- 全公司范围内人力负荷调配，尽量避免或减轻局部人力不足或闲置状况

## 未签 / 前期项目

**未签项目**：CRB3 审批通过且确认签署合同前的项目
- 项目号带部门号
- 项目管理功能的子功能，用于承载前期项目号阶段的项目执行

**前期项目**：同未签项目。

---

## 商务审查 CRB

**商务审查委员会（Commercial Review Boards）**：为评估项目实施中的项目采购出项合同风险，审核必要的规避计划和措施，确保公司进项合同的预计盈利计划和目标的实现的项目采购出项合同审核流程。

### CRB 阶段划分

| 阶段/模块 | CRB名称 | 目的 | 备注 |
| --- | --- | --- | --- |
| **进项合同模块** | | | |
| 第一阶段 CRB1 | 项目机会审批 | 审批是否立项追踪项目机会 | 适用于所有项目（除框架协议下的工作单） |
| 第二阶段 CRB2 | 投标报价审批 | 审批拟提交的标书（报价、合同条款等） | 适用于报价金额 > 50 万的项目及所有 EPC 项目 |
| 第三阶段 CRB3 | 进项合同审批 | 审批进项合同（签约价格、合同条款等） | 适用于所有项目（含工作单） |
| **进项合同变更模块** | | | |
| 第四阶段 CRB4 | 变更报价审批 | 审批拟提交的变更报价、合同条款等 | 适用于合同额变更金额 > 50 万元及所有 EPC 变更 |
| 第五阶段 CRB5 | 合同变更审批 | 批准签署变更合同及订单 | 适用于所有合同变更登记 |

### 成本计算

- 不含税报价金额 = 毛利 + 项目成本
- 项目成本不含税
- 含税报价金额 = 不含税报价金额 + 税费
- 税费 = 不含税报价金额 × 6%
- 股东利润 = 不含税报价金额 × 8%
